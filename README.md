## Installation

Node.js version used: 15.11.0.  
Npm version used: 7.19.1.

```sh
npm install 
```

## Dev server

```sh
npm run dev
```

## Build and preview locally

**Step 1:** Build:

```sh
npm run build
```

**Step 2:** Run local static server:

```sh
npm run serve
```
